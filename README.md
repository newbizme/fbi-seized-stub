# 🚨 FBI Seizure Notice - Сайт-заглушка

Высококачественная интерактивная статической страница-заглушка, имитирующая конфискацию домена федеральными ведомствами США (FBI, DOJ, HSI, USPIS).

![FBI Seizure Preview](https://raw.githubusercontent.com/fbi/seizure-badge.png) *Примечание: Заглушка полностью автономна, адаптивна и готова к деплою.*

## ✨ Особенности

- **Высококачественная векторная графика (SVG):** Все печати ведомств отрисованы на чистом SVG. Они отлично масштабируются и весят минимум.
- **Интерактивные CRT/Scanline эффекты:** Тонкие линии сканирования ретро-монитора и красноватое свечение из центра экрана.
- **Эффект Глитча (Glitch):** Анимированный глитч-эффект на главном предупреждении "THIS DOMAIN HAS BEEN SEIZED".
- **Динамический сбор данных (для реализма):**
  - Скрипт автоматически определяет текущий домен и подставляет его в форму ордера.
  - Определяет операционную систему пользователя.
  - Динамически запрашивает публичный IP-адрес посетителя через API `ipify.org` и выводит грозное предупреждение `WARNING: ALL CONNECTIONS ARE BEING LOGGED`.
- **Высокий уровень безопасности:** В комплекте идет `vercel.json` с настроенными заголовками безопасности (CSP, HSTS, X-Frame-Options, X-Content-Type-Options).
- **Мгновенная загрузка:** Нет внешних тяжелых JS-библиотек или фреймворков. Чистый HTML/CSS.

## 🚀 Развертывание на Vercel (в 2 клика)

### Способ 1: Импорт через панель управления Vercel

1. Выгрузите этот проект в свой новый репозиторий на GitHub.
2. Перейдите в панель управления [Vercel](https://vercel.com).
3. Нажмите **"Add New..."** ➔ **"Project"**.
4. Выберите ваш репозиторий из списка и нажмите **"Import"**.
5. Нажмите **"Deploy"**. Готово! Vercel автоматически распознает проект как статический сайт.

### Способ 2: Кнопка быстрого деплоя (Deploy Button)

Вы можете использовать кнопку ниже, предварительно заменив ссылку на ваш созданный репозиторий:
